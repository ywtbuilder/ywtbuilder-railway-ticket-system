
## 核心技术决策

1. 单体架构优先：先保证演示稳定性与可复现性，再逐步拆分。
2. 内嵌 H2：降低环境依赖，便于招聘方本地快速验证。
3. JWT 鉴权：前后端分离下维持会话一致性。
